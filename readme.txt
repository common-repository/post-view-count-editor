With Post View Count Editor you can edit the views of any of your post type, easily in your backend only if you admin

== Description ==

Add a Metabox in post edit page that shows the current Post View Count and let you to update the value.

== Installation ==

1. Unzip the file and upload the entire `post-view-editor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the edit post page you will see on the right side a box 'Post View Count Editor' 
4. Change the Value in the box and update the post as normal update !!
